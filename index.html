N-memory: WebApp

N-Memory was inspired by the game n-back, which was created in an effort to improve working memory in adults. My version was created to stretch users' mental math skills, focus as well as one's working memory. N-Memory was originally created as a webapp for my students in Mississippi.

Find it here.

Sushi-Game: Desktop App

This Sushi Game was a school project built with very basic Java user interface. In this game, the player prepares a plate of sushi with various ingredients that determine the plate's worth. It is placed on the sushi belt as customers buy or ignore the meal. There are artificial characters who are also trying to sell as much sushi as possible. If a plate stays on the belt for more than 2 turns without being bought, the plate is considered spoilt and thrown off the belt. The Scoreboard current winners and runner-offs.

Find it here.
Written in Java



QuickWeather: WebApp

QuickWeather was written in python with django to practice gathering data from external sites. It is a simple web app. It asks for a city, or a city and a country if there is more than one of the given city. The program references a weather API, OpenWeatherMap, and outputs the main weather with a brief description, and the temperature in celsius. Not Fahrenheit, because we all use Celsius on the other side of the world.

Find it here.
Written in HTML, CSS & Django (Python).



RealStockIndia: Java

RealStockIndia gathers names of Indian stocks and shows their prices. It uses web scraping, running through thousands of lines of html code from finance sites and uses a very simple intelligence to decipher which data (name of the company, and price) to display. It also allows the user to store the info it has scraped in a hashmap class called “StockStore”, so if needed again, can be loaded in O(1) time. I plan to improve this with an already created finance API so it retrieves information a lot faster than it does at the moment.

Find it here.
Written in Java
